It's important to note that sticky requires a bit of developer input to work properly. To create a sticky nav bar like this one, set the nav container width to 100%. It's also important to set the minimum top-anchor point to 1px, otherwise it'll never stick!

Here we're also having a bit of fun with transitions using event triggers from the sticky plugin. When the nav bar becomes sticky, it emits sticky.zf.stuckto:top, which we're listening to. Then we're adding a class to reduce the height and top padding of our nav bar. I'm not especially talented with design, but you get the gist of what you can do.
